## 🌐 قائمة الـ Endpoints

### المصادقة
```
✅ POST /api/auth/register     - إنشاء حساب جديد
✅ POST /api/auth/login        - تسجيل الدخول
✅ GET /api/auth/me            - الحساب الحالي
```

### المستخدمين
```
✅ GET /api/users/me           - الملف الشخصي
✅ PUT /api/users/me           - تحديث الملف
```

### الوكالات
```
✅ GET /api/agency/profile     - ملف الوكالة
✅ POST /api/agency/profile    - تحديث الملف
✅ GET /api/agency/listings/mine - إعلاناتي
✅ POST /api/agency/listings   - إنشاء إعلان
✅ PUT /api/agency/listings/{id} - تعديل إعلان
```

### الإعلانات
```
✅ GET /api/listings           - جميع الإعلانات
✅ GET /api/listings/{id}      - تفاصيل الإعلان
✅ GET /api/listings/{id}/similar - إعلانات مشابهة
✅ POST /api/listings/search/visual - بحث بالصورة
✅ GET /api/listings/semantic-search - بحث نصي
✅ POST /api/listings/seed-test-data - بيانات اختبار
```

### الدردشة
```
✅ GET /api/chat/conversations - المحادثات
✅ GET /api/chat/conversations/{id}/messages - الرسائل
✅ POST /api/chat/conversations/{id}/messages - إرسال
```

### الطلبات
```
✅ POST /api/requests          - إنشاء طلب
✅ GET /api/requests/mine      - طلباتي
```

### الإشعارات
```
✅ GET /api/notifications      - الإشعارات
✅ POST /api/notifications/{id}/read - قراءة
```

### لوحة التحكم
```
✅ GET /api/admin/agencies/pending - وكالات معلقة
✅ POST /api/admin/agencies/{id}/review - مراجعة وكالة
✅ GET /api/admin/listings/pending - إعلانات معلقة
✅ POST /api/admin/listings/{id}/review - مراجعة إعلان
✅ GET /api/admin/requests/pending - طلبات معلقة
✅ POST /api/admin/requests/{id}/review - مراجعة طلب
```

## 📝 أمثلة الاستخدام

### التسجيل
```typescript
authService.register({
  fullName: "أحمد محمد",
  phone: "+201001234567",
  email: "ahmed@example.com",
  password: "SecurePass123",
  nationalId: "12345678901234",  // يجب أن يكون 14 رقم بالضبط
  role: "Customer"  // أو "Agency"
});
```

### تسجيل الدخول
```typescript
authService.login("+201001234567", "SecurePass123");
// يتم حفظ الـ Token تلقائياً في localStorage
```

### إنشاء إعلان
```typescript
agencyService.createListing({
  type: "car",
  title: "تويوتا كورولا 2024",
  description: "سيارة بحالة ممتازة",
  brand: "Toyota",
  modelOrPartName: "Corolla",
  condition: "excellent",
  city: "القاهرة",
  area: "المقطم",
  year: 2024,
  kmsDriven: 5000,
  fuel: "petrol",
  transmission: "automatic",
  assembly: "japanese",
  price: 450000
});
```

### البحث النصي
```typescript
listingsService.semanticSearch("سيارة تويوتا رخيصة", 10);
```

### البحث بالصورة
```typescript
listingsService.visualSearch(imageFile, "car", 5);
```

### الدردشة
```typescript
// الحصول على المحادثات
chatService.getConversations();

// إرسال رسالة
chatService.sendMessage(conversationId, "مرحبا");
```

### المراجعة (Admin)
```typescript
// مراجعة وكالة
adminService.reviewAgency(agencyId, {
  approved: true,
  reason: "المستندات صحيحة"
});

// مراجعة إعلان
adminService.reviewListing(listingId, {
  approved: true,
  reason: "الإعلان يتبع القواعس"
});

// مراجعة طلب
adminService.reviewRequest(requestId, {
  approved: true,
  reason: "الطلب صحيح"
});
```
